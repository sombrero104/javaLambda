# 람다식(Lambda Expressions)
>식별자 없이 실행 가능한 함수 표현식.
<br/>
자바 8의 가장 특징적인 기능.<br/>
불필요한 코드를 줄이고 가독성을 향상시키는 것이 목적.<br/>

### Lambda Expressions Example
<pre>
( parameters ) -> expression body
( parameters ) -> { expression body }
() -> { expression body }
() -> expression body
</pre>
